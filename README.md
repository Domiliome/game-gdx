# Tower Defense Game (LibGDX)

Простой и производительный двумерный Tower Defense для платформ **Android**. Проект разработан на языке Java с использованием игрового фреймворка **LibGDX** и построен на чистой объектно-ориентированной архитектуре (ООП) с разделением логики, данных и отрисовки.

---

## 🕹️ Текущие игровые механики

*   **Движение по вейпоинтам:** Враги плавно перемещаются по заранее заданным точкам маршрута независимо от FPS (используется `deltaTime`).
*   **ИИ Башен (Поиск цели):** Башня автоматически сканирует карту и выбирает первого вошедшего в её радиус действия врага. Если враг погибает или уходит из радиуса, башня ищет новую цель.
*   **Самонаводящиеся снаряды:** Выпущенные снаряды летят точно за своей целью, пересчитывая вектор направления каждый кадр.
*   **Привязка к сетке (Grid Snapping):** Башни автоматически центрируются по клеткам при клике мышкой или тапе по экрану.
*   **Система улучшений:** При повторном клике на уже построенную башню запускается механика апгрейда, увеличивающая её параметры (урон, радиус, скорость атаки) по формуле.
*   **Отрисовка UI и здоровья:** Рендерер автоматически рисует динамические полоски здоровья (HP bars) над каждым врагом, линии дорог и радиусы действия башен.

## 🚀 Команды для сборки и запуска через консоль

Запускать команды необходимо из корневой папки проекта.

### 🖥️ Запуск на ПК (Desktop)
Компиляция общей логики и запуск игры в оконном режиме на компьютере (через LWJGL3):
```bash
./gradlew lwjgl3:run
```
*(Если ваш пусковой модуль на ПК называется по-старому, используйте `./gradlew desktop:run`)*

### 📱 Запуск на Android
Перед запуском убедитесь, что ваш смартфон подключен к ПК по USB, а в настройках устройства включена «Отладка по USB».

**Полная сборка и запуск на устройстве:**
```bash
./gradlew android:installDebug android:run
```

**Очистить кэш и принудительно пересобрать проект (при ошибках сборки):**
```bash
./gradlew clean assembleDebug
```
